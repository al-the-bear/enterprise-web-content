class TomReflectionInfo

Provides reflection-based operations for object serialization and deserialization.

This class is the main entry point for reflection operations in the Tom framework. It provides methods to: - Create instances of annotated classes - Set values on objects from JSON/Maps - Get values from objects as JSON/Maps - Resolve element types from collections

Key Features

Instance Creation

Create new instances of annotated classes using the default constructor or from JSON data:

final person = tomReflectionInfo.createInstance<Person>();
final loaded = tomReflectionInfo.createInstanceFromJson<Person>(mirror, jsonData);

Value Setting (Deserialization)

Set field values on objects from a map, handling: - Primitive types (int, double, bool, String) - DateTime and TomZoned* date/time types - Nested objects (annotated with @tomReflector) - Collections (List, Map) of primitives or annotated objects 

tomReflectionInfo.setValues(person, {'name': 'John', 'age': 30});

Value Getting (Serialization)

Extract all field values from an object as a JSON-compatible map:

final json = tomReflectionInfo.getValues(person);
final jsonString = tomReflectionInfo.convertToJsonString(person);

Debug Switches

Enable debug logging for troubleshooting reflection operations: - [debugSwitchLogReflectionDetailsMembers]: Log class member information - [debugSwitchLogReflectionDetailsValues]: Log value setting operations - [debugSwitchLogReflectionDetailsTypes]: Log type resolution details

ClassMirror<dynamic> mapValueElementType(Object collection)

Extracts the value type's [ClassMirror] from a Map type annotation.

Given a Map<K, V> collection, this method extracts the type parameter V and returns its corresponding [ClassMirror].

Example:

Map<String, Person> people = {};
final valueMirror = tomReflectionInfo.mapValueElementType(people);
// valueMirror is the ClassMirror for Person

Throws [TomReflectorException] if the value type is not found among annotated classes.

ClassMirror<dynamic> singleElementType(Object collection)

Extracts the element type's [ClassMirror] from a single-type-parameter collection (e.g., List, Set).

Given a List<T> or Set<T> collection, this method extracts the type parameter T and returns its corresponding [ClassMirror].

Example:

List<Person> people = [];
final elementMirror = tomReflectionInfo.singleElementType(people);
// elementMirror is the ClassMirror for Person

Throws [TomReflectorException] if the element type is not found among annotated classes.

ClassMirror<dynamic> _findClassMirror(String typeName, String errorKey, String errorMessage)

Finds a [ClassMirror] by type name among annotated classes.

Searches through all classes annotated with @tomReflector to find one matching the given [typeName].

Throws [TomReflectorException] with the provided [errorKey] and [errorMessage] if not found.

T convertFromJsonString<T extends Object>(T o, String s)

Creates an instance of type [T] from a JSON string.

This method parses the JSON string and populates the fields of the provided object [o] with the parsed values.

Example:

final person = Person();
tomReflectionInfo.convertFromJsonString(person, '{"name": "John", "age": 30}');
print(person.name); // "John"

Returns the same object [o] with fields populated.

T? createInstanceFromJsonString<T extends Object>(ClassMirror<dynamic> objectMirror, String s)

Creates a new instance of type [T] from a JSON string.

Uses the provided [ClassMirror] to instantiate the object, then populates it from the parsed JSON.

Returns the new instance, or null if creation fails.

T? createInstanceFromJson<T extends Object>(ClassMirror<dynamic> objectMirror, Map<String, Object?> jsonData)

Creates a new instance of type [T] from JSON data.

This method first attempts to use a fromJson constructor if available. If not, it creates an instance using the default constructor and populates fields via [setValues].

Example:

final mirror = tomReflector.reflectType(Person) as ClassMirror;
final person = tomReflectionInfo.createInstanceFromJson<Person>(
  mirror,
  {'name': 'Jane', 'age': 25},
);

Returns the new instance, or null if creation fails.

T createInstance<T extends Object>([Type? type])

Creates a new instance of type [T] using its default constructor.

The type must be annotated with @tomReflector and have a default (no-argument) constructor.

Example:

final person = tomReflectionInfo.createInstance<Person>();
// Or with explicit type:
final person = tomReflectionInfo.createInstance(Person);

Throws [TomReflectorException] if instantiation fails.

T setValues<T extends Object>(T o, Map<String, dynamic> values)

Sets values on an object from a map of field names to values.

This is the main entry point for deserialization. The method handles:

  • TomClass objects: Delegates to TomClass.setValues() for observable

field handling - Objects with jsonDecode method: Invokes that method directly - Other objects: Sets fields via reflection

Supported Field Types

  • Primitive types: int, double, bool, String, null
  • DateTime types: DateTime, TomZonedDate, TomZonedTime, TomZonedDateTime
  • Nested objects: Any class annotated with @tomReflector
  • Collections: List<T> and Map<String, T> where T is a supported type

Example:

final person = Person();
tomReflectionInfo.setValues(person, {
  'name': 'John',
  'age': 30,
  'address': {'street': '123 Main St', 'city': 'NYC'},
  'tags': ['developer', 'dart'],
});

Returns the same object [o] with fields populated.

String convertToJsonString(Object o)

Converts an object to a JSON string.

This is a convenience method that calls [getValues] and encodes the result to JSON.

Example:

final person = Person()
  ..name = 'John'
  ..age = 30;
final json = tomReflectionInfo.convertToJsonString(person);
// '{"name":"John","age":30}'
Map<String, Object?> getValues(Object o)

Gets all values from an object as a JSON-compatible map.

This is the main entry point for serialization. The method handles:

  • TomClass objects: Delegates to TomClass.toJson()
  • Objects with jsonEncode getter: Uses that getter's result
  • Other objects: Extracts non-static, non-final, non-private fields via reflection

Supported Field Types

  • Primitive types: returned as-is
  • DateTime types: converted to ISO 8601 strings
  • TomZoned* types: converted to string representation
  • Nested objects: recursively converted via [getValues]
  • Collections: elements are recursively converted

Example:

final person = Person()
  ..name = 'John'
  ..age = 30
  ..address = Address()..city = 'NYC';
final values = tomReflectionInfo.getValues(person);
// {'name': 'John', 'age': 30, 'address': {'city': 'NYC'}}

class TomReflector

class TomReflector extends Reflection

A reflector configured with capabilities required for JSON serialization and deserialization of annotated classes.

This reflector provides the following capabilities: - invokingCapability: Call methods and constructors via reflection - typingCapability: Access type information at runtime - typeRelationsCapability: Navigate class hierarchies - reflectedTypeCapability: Get actual Dart Type objects - typeAnnotationDeepQuantifyCapability: Access nested type annotations - typeAnnotationQuantifyCapability: Access type annotations - metadataCapability: Access annotations and metadata - superclassQuantifyCapability: Access superclass information

Usage:

@tomReflector
class MyClass {
  String name = '';
  int age = 0;
}

Classes must be annotated with @tomReflector to be discoverable by the reflection system at runtime.

class TomReflectorException

class TomReflectorException extends TomException

Exception thrown when reflection operations fail.

This exception is used throughout the reflection system to indicate errors during: - Instance creation from JSON - Field value setting via reflection - Type resolution for collections - Serialization/deserialization operations

Example:

try {
  final instance = tomReflectionInfo.createInstance<MyClass>();
} on TomReflectorException catch (e) {
  print('Reflection failed: ${e.defaultUserMessage}');
}
