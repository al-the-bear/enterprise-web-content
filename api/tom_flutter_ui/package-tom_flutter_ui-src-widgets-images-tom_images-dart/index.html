class TomIcon

class TomIcon extends TomImageBase

Tom wrapper for [Icon].

Adds ID, group, authorization, and resource resolution for icon and alt.

Drop-in replacement for Flutter's [Icon] widget:

TomIcon(icon: Icons.save)           // Explicit icon
TomIcon(icon: Icons.save, size: 24) // With size

When icon is not provided, resolves via resource lookup: 1. Gets icon name from resources using suffix 'icon' 2. Looks up [IconData] from [TomUIIcons] provider

Example resource entry:

w-saveButton.icon: "save"
w-saveButton.alt: "Save the document"

Constructors

TomIcon.new({IconData? icon, Key? key, String? tomId, InvalidType tomGroup, InvalidType scope, InvalidType authorizer, InvalidType authorizerStrategy, InvalidType uiStateController, double? size, double? fill, double? weight, double? grade, double? opticalSize, Color? color, List<Shadow>? shadows, String? semanticLabel, TextDirection? textDirection, bool? applyTextScaling})

Properties

Creates a TomIcon widget.

Use [icon] for explicit icon, or omit for resource resolution:
