class TomSnackBarAction

class TomSnackBarAction extends SnackBarAction

Tom wrapper for [SnackBarAction] with construction-time resource resolution.

Extends [SnackBarAction] directly to fit in SnackBar(action:).

Resources are resolved at construction time: - [label] — if null, resolved from $tomId.label - [textColor] — resolved from $tomId.textColor - [disabledTextColor] — resolved from $tomId.disabledTextColor 

TomSnackBar(
  tomId: 'saveSuccess',
  action: TomSnackBarAction(
    tomId: 'undo',  // Resolves label from resources
    onPressed: () => handleUndo(),
  ),
)

Constructors

If [label] is null and [tomId] is provided, resolves the label from resources at $tomId.label.
