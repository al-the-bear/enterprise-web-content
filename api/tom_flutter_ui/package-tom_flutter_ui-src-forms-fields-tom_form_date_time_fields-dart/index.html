class TimeRange

class TimeRange

A (start, end) pair of [TimeOfDay] values, used as the value type for time-range fields. Equatable so observers see a real change only when at least one endpoint moves.

No concrete widget ships in Phase 3 (the spec table treats Phase 3 concretes as a "target — not every row is required"). Apps that need a time-range picker today can subclass [TomFormTimeRangeField] with two stacked [TomFormTimePicker]-style buttons.

class TomFormDateTimeRangeField

abstract class TomFormDateTimeRangeField extends TomField<DateTimeRange<DateTime>>

Abstract field for a (start, end) pair of full [DateTime] values. Reuses Flutter's [DateTimeRange] as the value type. As with [TomFormTimeRangeField], no concrete widget ships in Phase 3 — apps that need one can compose [TomFormDateTimePicker] twice.
