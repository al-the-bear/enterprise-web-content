class TomPopupMenuItem

class TomPopupMenuItem<T> extends PopupMenuItem<T> with TomWidgetMixin

Tom widget that EXTENDS [PopupMenuItem] with authorization and resources.

This widget IS a [PopupMenuItem] — it passes `is PopupMenuItem` checks and `is PopupMenuEntry` checks for use in [showMenu] and [PopupMenuButton].

**StatefulWidget pattern:** - Widget has the mixin and `tomId` - State overrides `build()` and accesses mixin methods through `widget` 

PopupMenuButton<String>(
  itemBuilder: (context) => [
    TomPopupMenuItem<String>(tomId: 'edit', value: 'edit', child: Text('Edit')),
    TomPopupMenuItem<String>(tomId: 'delete', value: 'delete', child: Text('Delete')),
  ],
)
