class TomStyleResolvers

Phase 3: Complex style resolution helpers for Tom UI widgets.

Provides resolution of complex Flutter objects (TextStyle, ButtonStyle, InputDecoration, MenuStyle, ShapeBorder, BorderRadius, Decoration, BoxConstraints, AlignmentGeometry, BorderSide, Offset, TableBorder) from JSON resource maps.

These helpers are separate from [TomNodeBase] because they require Material library imports.

Usage in widgets:

final style = this.style ?? resolveButtonStyleOrNull(context, 'style');

static TextStyle? parseTextStyle(Map<String, dynamic>? map)

Resolves a [TextStyle] from a resource map.

The resource map can contain: - fontSize (double): Font size in logical pixels - fontWeight (String): "normal", "bold", "w100"-"w900" - fontStyle (String): "normal", "italic" - color (String): Color name or hex value - backgroundColor (String): Color name or hex value - letterSpacing (double): Letter spacing - wordSpacing (double): Word spacing - height (double): Line height multiplier - decoration (String): "none", "underline", "overline", "lineThrough" - decorationColor (String): Color name or hex value - decorationStyle (String): "solid", "double", "dotted", "dashed", "wavy" - overflow (String): "clip", "fade", "ellipsis", "visible"

Returns null if no map is found or parsing fails.

Resolves a [ButtonStyle] from a resource map.

All color properties are wrapped in [WidgetStatePropertyAll] for simplicity. For state-dependent colors, use explicit [ButtonStyle] in code.
